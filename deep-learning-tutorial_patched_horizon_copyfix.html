💡 핵심 개념

ANN(Artificial Neural Network)은 인간의 뇌 구조를 모방한 기본적인 딥러닝 모델입니다.

  • 입력층: 데이터를 받아들이는 층
  • 은닉층: 특징을 학습하는 층
  • 출력층: 최종 결과를 출력하는 층

🔄 순전파 (Forward Propagation)

⬅️ 역전파 (Backpropagation)

순전파 계산:
y = activation(Wx + b)
W: 가중치, x: 입력, b: 편향, activation: 활성화 함수

4. LSTM & GRU

🧠 LSTM 셀 구조

LSTM의 내부 구조: Cell State와 Gates

🔀 GRU 구조

GRU: LSTM의 간소화된 버전

📈 LSTM vs GRU 비교

장기 의존성 학습 성능 비교

구성요소 LSTM GRU
게이트 수 3개 (Forget, Input, Output) 2개 (Reset, Update)
메모리 셀 별도의 Cell State 존재 Hidden State만 사용
파라미터 수 많음 상대적으로 적음
학습 속도 느림 빠름
성능 복잡한 패턴 학습에 유리 간단한 패턴에서 효율적
LSTM 주요 수식:
f_t = σ(W_f · [h_t-1, x_t] + b_f) // Forget Gate
i_t = σ(W_i · [h_t-1, x_t] + b_i) // Input Gate
C̃_t = tanh(W_C · [h_t-1, x_t] + b_C) // Candidate values
C_t = f_t * C_t-1 + i_t * C̃_t // Cell State
o_t = σ(W_o · [h_t-1, x_t] + b_o) // Output Gate
h_t = o_t * tanh(C_t) // Hidden State

💡 시계열 예측에서의 Attention

LSTM에 Attention을 추가하면 중요한 시점에 더 집중할 수 있습니다.

  • 장기 의존성 문제 해결
  • 해석 가능한 모델 (어느 시점이 중요한지 시각화 가능)
  • 성능 향상 (특히 긴 시퀀스에서)

LSTM with Attention 구현

import tensorflow as tf
from tensorflow.keras import layers, Model

class AttentionLayer(layers.Layer):
    def __init__(self, units):
        super(AttentionLayer, self).__init__()
        self.W = layers.Dense(units)
        self.V = layers.Dense(1)
        
    def call(self, values):
        # values shape: (batch_size, seq_len, hidden_dim)
        
        # Attention 점수 계산
        score = tf.nn.tanh(self.W(values))
        attention_weights = tf.nn.softmax(self.V(score), axis=1)
        
        # 가중 평균 계산
        context_vector = attention_weights * values
        context_vector = tf.reduce_sum(context_vector, axis=1)
        
        return context_vector, attention_weights

# LSTM with Attention 모델
def build_lstm_attention_model(input_shape):
    inputs = layers.Input(shape=input_shape)
    
    # LSTM 레이어 (return_sequences=True 필수!)
    lstm_out = layers.LSTM(128, return_sequences=True)(inputs)
    lstm_out = layers.Dropout(0.2)(lstm_out)
    
    # Attention 레이어
    attention = AttentionLayer(128)
    context_vector, attention_weights = attention(lstm_out)
    
    # 출력 레이어
    outputs = layers.Dense(64, activation='relu')(context_vector)
    outputs = layers.Dropout(0.2)(outputs)
    outputs = layers.Dense(1)(outputs)
    
    model = Model(inputs=inputs, outputs=outputs)
    return model

# 모델 생성 및 컴파일
model_with_attention = build_lstm_attention_model((50, 1))
model_with_attention.compile(
    optimizer='adam',
    loss='mse',
    metrics=['mae']
)
